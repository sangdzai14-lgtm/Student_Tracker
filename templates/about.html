About This System

University Course Performance Tracker & Grade Prediction System

Project ID

TEC004/05

Project Description

A comprehensive Python application for managing student academic records, processing grade data from various file formats, storing in a database, performing statistical analysis on academic performance, and using machine learning to predict final grades.

Key Features
  • OOP Grade Management: Abstract Person class hierarchy with Student and Instructor
  • Multiple Grading Schemes: Weighted, curved, and pass/fail grading systems
  • File I/O: Import grades from CSV files, export reports to JSON
  • SQLite Database: Comprehensive relational database for students, courses, grades
  • Data Analytics: Pandas-based analysis with grade distributions and correlations
  • Data Visualization: MatPlotLib charts for grade analysis
  • AI Grade Prediction: Linear Regression model for predicting final grades
  • At-Risk Student Alerts: Automatic identification of students needing support
  • Web Portal: Flask-based web interface for data entry and reporting
Technology Stack
Backend
  • Python 3.8+
  • Flask Web Framework
  • SQLite Database
  • Pandas Data Analysis
  • Scikit-learn ML
Frontend

  • Bootstrap 5 CSS
  • Chart.js Visualization

Learning Outcomes

Development Phases
